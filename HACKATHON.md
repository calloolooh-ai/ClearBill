# Lumora Hacks Summer 2026

ClearBill is built for [Lumora Hacks Summer 2026](https://lumora-hacks-2026.devpost.com/).

## Theme

"Build anything" — an open-ended hackathon with no restriction on project type or tech stack.

## Eligibility

- Ages 13–17, students only (no companies/professional orgs)
- Open to all countries/territories (standard exceptions apply)
- Solo or team
- Open to beginners and experienced participants

## Key Dates

- **Hackathon start:** July 10, 2026, 7:30 AM EDT
- **Submission deadline:** July 10, 2026, 5:00 PM IST
- Projects must be newly created during the hackathon period.

## Submission Requirements

1. Public code repository (GitHub) with project code and assets
2. Demo video, 2–6 minutes, showing functionality and what's unique about it
3. Devpost project description: problem, solution, tech stack
4. Optional: live demo link

## Judging Criteria

| Criterion      | Weight |
|----------------|--------|
| Originality    | 25%    |
| Practicality   | 25%    |
| Presentation   | 50%    |

Presentation is weighted highest — the demo video and explanation quality matter as much as the build itself.

HowToHackathon, CodeCrafters

## How ClearBill Maps to Judging Criteria

- **Originality:** AI-explained billing transparency tool — combines OCR/PDF parsing, rule-based fee detection, and constrained LLM explanation (never invents charges) in one flow.
- **Practicality:** Solves a real, universal problem (confusing bills) with a working upload → dashboard pipeline; runs on real bills (PDF/PNG/JPG).
- **Presentation:** Apple-inspired UI, smooth motion, clear dashboard with charts — built to demo well in a 2–6 minute video.

## Deliverables Checklist for Submission

- [ ] Public GitHub repo (`ClearBill`) with clean commit history under the account submitting
- [ ] Live Vercel deployment link
- [ ] 2–6 minute demo video covering: upload → extraction → AI explanations → dashboard → fee alerts → comparison
- [ ] Devpost write-up: problem statement, solution, tech stack (Next.js 15, TypeScript, Tailwind, shadcn/ui, Framer Motion, Recharts, Groq)
